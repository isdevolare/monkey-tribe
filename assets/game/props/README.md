# Monkey Tribe Prop Assets

Transparent PNG props generated for the village/base-builder scene.

## Files

- `prop_campfire.png` - central village campfire
- `prop_crate.png` - supply crate decoration
- `prop_barrel.png` - storage barrel decoration
- `prop_log_pile.png` - wood/log resource decoration
- `prop_banana_basket.png` - banana resource decoration
- `prop_rope_coil.png` - crafting/utility decoration
- `prop_fence_segment.png` - wooden village fence segment
- `prop_training_dummy.png` - training nest / fighter training decoration

## Usage Notes

- All files are PNGs with alpha transparency.
- Designed for mobile portrait RTS/base-builder readability.
- Style target: cartoon 3D jungle, warm lighting, playful monkey tribe village.
- Best rendered as absolute-positioned props over `bg_jungle_game.png`.
- Keep sizes varied in-game so they feel decorative, not like full buildings.
